어렸을 때부터 흥미가 생기는 무언가를 만드는 것을 좋아했습니다. 그러다보니 새로운 기술에도 관심이 많았습니다.

시각디자인과 졸업 후, 디자인과 웹 기술의 접목된 웹 퍼블리셔(웹 프론트 엔드)가 저에게 새롭게 다가왔습니다.

웹에 관련된 자료와 교육을 통하여 기술을 습득하게 되었고, 퍼블리셔에서 웹 프론트 엔드 개발자로 나아가고 싶어졌습니다.

저의 손으로 직접 만든 사이트가 서비스된다면, 또 다양한 기술은 접 할 수 있다면 행복할 것 같습니다.

활발한 성격은 아니지만, 저의 부족한 부분을 채워 사람들과 소통을, 사용자에겐 콘텐츠를 전달하는 '웹 프론트 엔드'의 길을 걷고 싶습니다.
