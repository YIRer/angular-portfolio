포트폴리오 계획서

restFul Nodejs Angular(2) MongoDB

이름 접근 경로 설명
메인 GET / 메인페이지. 포트폴리오를 간단하게 리스트업
포트폴리오 GET /portfolio 포트폴리오페이지. 앨범 API와 게시판 API가 존재
자기소개 GET /profile 프로필페이지
연락처 GET /connect 연락처 제공. 가능하다면 차후에 이메일 전송기능 추가
앨범 GET /portfolio/album 앨범 어플리케이션 접근, 리스트 업. 우선 구현 후 차후에 로그인기능 부여
GET/POST /portfolio/album/new 글쓰기 기능
GET /portfolio/album/:id 상세보기
GET/PUT /portfolio/album/:id/edit 수정
DELETE /portfolio/album/:id 삭제
게시판 GET /portfolio/dashboard 게시판 어플리케이션 접근, 리스트 업. 우선 구현 후 차후에 로그인기능 부여
GET/POST /portfolio/dashboard/new 글쓰기 기능
GET /portfolio/dashboard/:id 상세보기
GET/PUT /portfolio/dashboard/:id/edit 수정
DELETE /portfolio/dashboard/:id 삭제
GET/POST /portfolio/dashboard/:id/comment/new 코맨트 생성
GET/PUT /portfolio/dashboard/:id/comment/:comment_id/edit 코맨트 수정
DELETE /portfolio/dashboard/:id/comment/:comment_id 코맨트 삭제
로그인 GET /portfolio/login 포트폴리오 어플리케이션 접근을 위한 로그인. 고정 아이디 제공. 차후에 구현
GET/POST /portfolio/register 포트폴리오 어플리케이션 접근을 위한 회원가입. 차후에 구현
